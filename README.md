# AI Memory - 向量记忆增强系统

为 ChatBox 提供长期记忆能力的 OpenAI 兼容网关。

个人小实验，希望让AI不要忘东西了。

## 功能

- OpenAI API 兼容接口
- 向量检索增强（RAG）
- DeepSeek 模型集成
- 本地向量存储（FAISS/Chroma）

## 技术栈

- Python 3.x
- FastAPI - Web 框架
- sentence-transformers - 嵌入模型
- FAISS/Chroma - 向量数据库
- httpx - HTTP 客户端
